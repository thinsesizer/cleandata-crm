CLEAN DATA CRM - UPLOAD INSTRUCTIONS
====================================

This folder contains the complete CleanData CRM application.

FILES TO UPLOAD TO GITHUB:
--------------------------
✓ All files in this folder (except this README)
✓ Keep the folder structure exactly as is
✓ src/, prisma/, mobile/ folders are important
✓ package.json, next.config.js, etc. are required

WHAT NOT TO UPLOAD:
-------------------
✗ node_modules/ folder (will be created automatically)
✗ .env.local file (contains secrets, create after upload)
✗ .next/ folder (build output)

UPLOAD METHOD:
--------------
1. Go to https://github.com/new
2. Repository name: cleandata-crm
3. Select: Public
4. Click "Create repository"
5. Click "uploading an existing file"
6. Drag and drop ALL files from this folder
7. Click "Commit changes"

AFTER UPLOAD:
-------------
Your repo will be at:
https://github.com/thinsesizer/cleandata-crm

Then you can deploy to Vercel for free hosting!
